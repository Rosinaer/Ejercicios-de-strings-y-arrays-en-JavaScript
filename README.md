# Ejercicios de Strings y Arrays en JavaScript

Este trabajo práctico contiene una serie de ejercicios de JavaScript, enfocados en el manejo de strings, array.

## Cómo usarlo

1. Abrir el archivo index.html en el navegador.
2. Abrir la consola (F12).
3. Los resultados de cada ejercicio se mostrarán allí.

## Archivos incluidos

- index.html
- main.js

## Tecnologías

- HTML5
- JavaScript
